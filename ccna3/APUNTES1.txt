COMANDOS PARA AGREGAR UNA DIRECCION IP EN UNA INTERFAZ DE RED DEL ROUTER:
	
	int nombre_interfaz (interface)
	desc mensaje o descripcion (descripcion)
	ip  add direccion_ip mascara_subred (ip)
	no shut 
	exit 

	ejemplo: 
		int g0/0
		desc interfaz de red del LAN router
		ip add 10.0.0.1 255.255.255.0
		no shut
		exit
		
	nota: la g significa gigabithert por que todas las interfaces del router son en gigabit 
	si fuera un swith fue una f por que las interfaces en los swith son fastethernet 0/1

comandos para implementar el enrutamiento estatico en routers cisco

en el modo de configuracion global:
opcion 1
ip route direccion-ip-red-destino mascara-subred 
destino direccion-ip-interfaz-router-vecino 
opcion 2
ip route direccion-ip-red-destino mascara-subred 
destino direccion-ip-interfaz-router-local

ejemplo: 
	opcion 1
	ip route 172.0.100.1 255.255.192.0 100.0.5.2
	opcion 2
	ip route 172.0.100.1 255.255.192.0 g0/1
